# Figures

Paper figures for the UXBench manuscript.

## Files

| File | Description |
|------|-------------|
| `fig_uxbench_overview.pdf` | Main benchmark overview diagram |
| `fig_task1.png` | Task 1 illustration |
| `fig_task1_result.jpg` | Task 1 results visualization |
| `fig_task2_timeline.pdf` | Task 2 evaluation timeline |
| `fig_signal_distribution.{pdf,png}` | Distribution of user feedback signals |
| `fig_human_annotation.pdf` | Human annotation agreement |
| `fig_uxbench_data_quality_human_test.pdf` | Data quality validation |
| `fig_grm_training_loss_redrawn.pdf` | GRM training loss curve |
| `fig_reasoning_efforts.pdf` | Reasoning effort ablation |
| `fig_ablation_*.pdf` | Various ablation study figures |
| `fig_task3_representative_strategy_distribution.pdf` | Recovery strategy distribution |
| `fig_prompt_templates.tex` | LaTeX source for prompt template figure |
| `scenarios/` | Per-scenario breakdown figures |

## Format

- Vector graphics: PDF (preferred for LaTeX inclusion)
- Raster graphics: PNG/JPG (for web display)

## Regenerating

Most figures are generated from the data in `../experiments/`. The plotting scripts are not included in this release but the raw data is available for re-plotting.
